# Atajos de Teclado para el Sistema de Gestión de Transporte

Este documento proporciona una lista de atajos de teclado útiles que pueden mejorar la eficiencia al utilizar el sistema de gestión de transporte. Familiarizarse con estos atajos puede ayudar a los usuarios a navegar más rápidamente y realizar tareas comunes sin necesidad de utilizar el ratón.

## Atajos Generales

- **Ctrl + N**: Crear un nuevo registro (vehículo, conductor, gasto, etc.)
- **Ctrl + S**: Guardar cambios en el registro actual.
- **Ctrl + Z**: Deshacer la última acción.
- **Ctrl + Y**: Rehacer la última acción deshecha.
- **Ctrl + F**: Abrir la barra de búsqueda para filtrar registros.

## Navegación

- **Ctrl + D**: Ir al Dashboard.
- **Ctrl + V**: Acceder a la sección de Vehículos.
- **Ctrl + C**: Acceder a la sección de Conductores.
- **Ctrl + M**: Acceder a la sección de Documentos.
- **Ctrl + G**: Acceder a la sección de Gastos.
- **Ctrl + T**: Acceder a la sección de Fletes.
- **Ctrl + P**: Acceder a la sección de Mapas.

## Gestión de Documentos

- **Ctrl + U**: Subir un nuevo documento.
- **Ctrl + R**: Refrescar la lista de documentos.

## Control de Gastos

- **Ctrl + E**: Iniciar la captura de un nuevo gasto.
- **Ctrl + A**: Abrir el reporte de gastos.

## Funciones del Dashboard

- **Ctrl + H**: Ver las alertas de documentos próximos a vencer.
- **Ctrl + L**: Ver estadísticas de gastos mensuales.

## Otros Atajos Útiles

- **F1**: Abrir la ayuda del sistema.
- **Esc**: Cerrar ventanas emergentes o diálogos.
- **Alt + Enter**: Ver detalles del registro seleccionado.

## Notas

- Los atajos pueden variar dependiendo de la configuración del sistema y del navegador utilizado.
- Es recomendable practicar estos atajos para mejorar la fluidez en el uso del sistema.
